# Campus One Frontend

Web frontend repository for Campus One.

## Apps

- `apps/web-lms` - public LMS, login entry, school registration, onboarding, and platform-facing flows.
- `apps/web-school` - tenant/internal school portal for admin, student, professor, applicant, and alumni workflows.

## Packages

- `packages/api-client` - shared typed API client.
- `packages/ui` - shared web UI components.
- `packages/shared-contracts` - frontend-safe DTOs and enums from backend contracts.
